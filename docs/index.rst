欢迎来到Flask
================

.. image:: _static/logo-full.png
   :alt: Flask: web development, one drop at a time
   :class: floatingflask

欢迎来到Flask.本文档分为几个部分.我推荐你从 :ref:`installation` 开始，然
后跳到 :ref:`quickstart`.除了快速上手外, :ref:`tutorial` 更详细的介绍了怎
样在Flask中创建一个完整的(依然微小)的应用程序.如果你想更深入的了解Flask
内部，查看阅 :ref:`api` 文档.在 :ref:`patterns` 章节描述了常见的模式.

Flask依赖于两个外部的库： `Jinja2`_ 模板引擎和 `Werkzeug`_ WSGI工具包.
这些库没有在这里列出文档。如果你想深入了解，查看以下链接：

-   `Jinja2 Documentation <http://jinja.pocoo.org/2/documentation/>`_
-   `Werkzeug Documentation <http://werkzeug.pocoo.org/documentation/>`_

.. _Jinja2: http://jinja.pocoo.org/2/
.. _Werkzeug: http://werkzeug.pocoo.org/
